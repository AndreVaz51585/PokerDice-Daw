# Poker Dice Game

This is the repository for my Web Application Development (DAW) university project. The goal was to build a full-stack web version of the classic Poker Dice game.

## Tech Stack
To build this, I developed a backend API and an interactive frontend using the following tools:

**Backend:**

- Kotlin + Spring Boot
- PostgreSQL + JDBI
- Server-Sent Events (SSE): To push game updates to the client in real time.

**Frontend:** 

- React + TypeScript
